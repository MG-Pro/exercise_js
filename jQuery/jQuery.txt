Домашнее задание к лекции 6.1 «Библиотека jQuery»
Реализуйте конвертор валют с использованием библиотеки jQuery. Пользователь должен ввести сумму, выбрать исходную и желаемую для конвертации валюты и получить результат вычислений.
В форме должно быть 4 поля:
* исходная валюта (select)
* сумма (input)
* новая валюта (select)
* сумма (input)
Логика работы формы:
1. В базовом состоянии пользователю должна быть видна и доступна только форма, все остальные блоки (если вы будете их использовать) должны быть скрыты.
2. После ввода суммы, выбора валют результат вычислений должен появляться в свободном поле ввода без перезагрузки страницы.
3. Пользователь должен иметь возможность незамедлительно произвести еще одну конвертацию без перезагрузки страницы.
4. Список валют и их курс хранится на сервере. Вы должны получать актуальную информацию на момент конвертации.
Оформление сделайте на ваше усмотрение.
Не забывайте о принципе DRY и, по возможности, переиспользуйте код, который вы уже написали.
Задание 1.
* Используйте для запроса к API url http://university.netology.ru/api/currency.
* Создайте страницу с формой, в которой будут все необходимые поля.
* Все вычисления должны происходить без перезагрузки страницы.
Задание 2. (не обязательное)
* Используйте для запроса к API url http://university.netology.ru/api/currency.
* Создайте страницу с формой, в которой будут все необходимые поля.
* Все вычисления должны происходить без перезагрузки страницы.
* Проверяйте ввод исходной суммы на соответствие формату - числу. При вводе букв форма любым образом (подсветка, ошибка) должна сообщать пользователю о неверном формате.
* Если ввести значение в поле производной суммы, не меняя при этом валюту, должна пересчитываться исходная сумма.
* Если в исходной валюте выбрать валюту, равную производной, то в поле производной валюты значение меняется на исходную и суммы меняются местами (обратное преобразование).
* Создать интерфейс для добавления собственной валюты и ее курса.
