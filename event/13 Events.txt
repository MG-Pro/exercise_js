Домашнее задание к лекции 4.4 «Обзор событий»

Необходимо выполнить следующие задания:

1. Список с актерами. Нужно реализовать возможность выделения. При левом клике мышкои выделять одного актера. При клике с зажатым CTRL выделять несколько актеров;

2. Поле "пароль". При вводе нужно считать количество символов. Если символов в пароле меньше 5, то выводить показывать сообщение под полем "Пароль должен быть длиннее 5 символов", если больше, то показывать "Хороший пароль”;

3. Блок “Информация”. При наведении курсора на этот блок подсвечивать его цветом. В момент когда убираете мышь, убирать подсветку;

4. Слайдер. Реализовать возможность двигать слайдер;

5. Кнопка Submit. При нажатии на кнопку выводить значения полей "Name, Last name, Password" в консоль. При нажатии на кнопку с зажатой клавишей Shift выводить значение полей в alert(). Действие по умолчанию для формы не должно срабатывать (страница не должна перезагружаться).
